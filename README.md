# captain

## Install

`go get github.com/senghoo/captain`

## Run

`captain web`

## Useful commands 

Add user 

`captain useradd user1 pass1 [-admin]`

else 
`captain -h`

## Environment

* DB_HOST
* DB_NAME
* DB_USER
* DB_PASSWORD
* GITHUB_CLIENT_ID
* GITHUB_CLIENT_SECRET
* CSRF_KEY
* CAPTAIN_WORKSPACE
* SITE_URL
